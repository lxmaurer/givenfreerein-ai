# Given Free Rein – National Geographic Style Horse Culture Journal

A fresh, editorial website redesign built from extracted content from the live Given Free Rein website. This project demonstrates a complete rebuild with National Geographic-inspired design aesthetics.

## 🎯 Project Overview

This website was built by:
1. **Crawling** the live Given Free Rein website (`www.givenfreerein.com.au`)
2. **Extracting** all content (articles, titles, excerpts, categories, metadata)
3. **Designing** a completely new National Geographic-style editorial layout
4. **Building** homepage, article pages, and navigation from scratch

## 📁 Project Structure

```
givenfreerein-ai/
├── index.html              # Homepage with hero, featured story, article grid
├── article-comanche.html   # Sample article page (Comanche story)
├── styles.css              # Complete CSS with National Geographic styling
├── content/
│   ├── articles.json       # Extracted article data
│   ├── categories.json     # Category and region information
│   └── site.json          # Site metadata and navigation
└── images/
    └── sidesaddle_sevilla.jpg  # Hero image
```

## 🎨 Design Features

### National Geographic-Inspired Aesthetics
- **Typography**: Crimson Text serif for body, Playfair Display for headings
- **Color Palette**: Warm brown accents (#8b7355) on cream backgrounds
- **Layout**: Editorial grid system with generous spacing
- **Navigation**: Fixed header with smooth backdrop blur
- **Cards**: Clean white cards with subtle shadows and hover effects

### Content Organization
- **Featured Story**: Prominent display of the Comanche article
- **Article Grid**: Three-column responsive layout for latest stories
- **Categories**: Cultures, Breeds, Gear & Tack, Destinations
- **Newsletter**: Integrated signup for "Join The Ride"
- **Footer**: Comprehensive site links and business information

## 📄 Extracted Content

### Articles Featured:
1. **Lords of the Southern Plains** (Comanche Horse Nation) - Featured story
2. **Escaramuza** (Mexican women's riding tradition)
3. **The Side Saddle** (Evolution of equestrian tack)
4. **The Horses of the Camargue** (French wetland horses)

### Categories:
- **Cultures**: Horse traditions and riding practices
- **Breeds**: Specific horse breeds and their cultural significance
- **Gear & Tack**: Equestrian equipment and its evolution
- **Destinations**: Horse travel and tourism locations

## 🚀 Getting Started

1. **Open** `index.html` in your browser to view the homepage
2. **Navigate** using the fixed header menu
3. **View articles** by opening `article-comanche.html`
4. **Customize** content by editing the JSON files in `/content/`

## 🎯 Key Design Principles

- **Editorial Authority**: Clean, sophisticated typography hierarchy
- **Visual Hierarchy**: Clear content organization with proper spacing
- **Accessibility**: High contrast support, keyboard navigation, screen reader friendly
- **Responsive**: Mobile-first design that works on all devices
- **Performance**: Optimized fonts, efficient CSS, semantic HTML

## 📱 Responsive Breakpoints

- **Desktop**: 1024px+ (full grid layouts)
- **Tablet**: 768px (stacked layouts, adjusted spacing)
- **Mobile**: 480px (single column, optimized touch targets)

## 🛠 Technical Implementation

- **CSS Custom Properties**: Consistent design system with CSS variables
- **Modern Layout**: CSS Grid and Flexbox for responsive layouts
- **Typography Scale**: Consistent sizing with clamp() for fluid typography
- **Smooth Interactions**: CSS transitions for hover states and navigation
- **Accessibility**: WCAG compliant with proper focus management

## 📊 Content Source

All content was ethically extracted from the live Given Free Rein website and restructured for this new design. The original website served only as a content library - the design is completely original and not derived from existing layouts.
