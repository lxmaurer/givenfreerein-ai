# Site Rules

1. Never redesign the homepage layout without instruction.

2. Prefer small CSS edits over HTML structure changes.

3. Only perform ONE change per task.

4. Never modify more than one section of the site in a single task.

5. Always verify changes visually on http://localhost:8000.

6. Preserve the editorial style of the site.

7. Do not change typography or fonts unless specifically requested.

8. Do not alter header or navigation layout unless a task explicitly says so.

9. Avoid large layout changes like flex/grid rewrites unless instructed.

10. If unsure, do not execute the change.
